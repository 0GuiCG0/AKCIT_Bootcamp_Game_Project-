# AKCITBootcamp - Doomed Village

## Doomed Village:
### Projeto desenvolvido durante o Bootcamp da AKCIT na UFG - Campus Samamaia. Feito quase totalmente com IA, para imersão e compreensão dos limites e capacitar os participantes a entenderem a se tornarem curadores de artes e codigos produzidos pela(s) IA(s) utilizadas.
### Nem todos os efeitos e sprites foram feitos por IA, multiplos foram encontrados no itch.io
